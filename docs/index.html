网页 API 使用说明

欢迎使用 酷乐短剧 / 影视资源聚合搜索 API。

您可以通过访问以下 URL 参数直接调用搜索 API：

https://selfemo.github.io/ShortDramaSearch/?q=关键词&from=来源&format=json

参数说明：

  • q / name / s：搜索关键词（可中英文），支持空格拆分
  • from：搜索来源，默认 all，可选：
    • 聚合类：all, netdisk
    • 网盘类：baidu, quark, aliyun, tianyi, uc, mobile, 115, pikpak, xunlei, 123
    • 链接类：magnet, ed2k
  • format：输出格式，json 为纯 JSON，其他值为网页模式

示例 API 调用：

https://selfemo.github.io/ShortDramaSearch/?q=飞驰人生3&from=netdisk&format=json

API 将返回如下 JSON 数据示例：

{
  "code": 0,
  "msg": "success",
  "keyword": "飞驰人生3",
  "type": "netdisk",
  "total": 6,
  "data": { ... }
}
